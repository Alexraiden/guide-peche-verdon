## Instructions de remplacement

1. **Préparer vos photos** au bon format (JPG ou WebP)
2. **Renommer** vos photos avec les noms exacts listés ci-dessus
3. **Placer** les fichiers dans les bons dossiers
4. **Optimiser** pour le web (compression sans perte de qualité visible)
5. **Dimensions recommandées:**
   - Hero images: 1920x1080px minimum
   - Poissons: 800x600px
   - Lacs: 1200x800px
   - Guide: 800x800px

## Alternative: Utilisation des URLs directes
Pour l'instant, le site utilise des URLs Unsplash directement dans le code HTML.

**Pour la production:**
- Téléchargez les images depuis Unsplash (libres de droits)
- OU remplacez par vos propres photos
- Placez-les dans le dossier `images/` correspondant
- Mettez à jour les chemins dans les fichiers HTML
